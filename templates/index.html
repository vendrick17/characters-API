Random character generation


This option allows you to generate random black and white silhouettes as well as colored characters, to ensure the quality of the output we project a random image from our dataset into the latent space, this will lead to an improvement of the quality of the characters, allow the appereance of more features and avoid completely random images in the output.

Guided Character creation


By using this tool you will be able to generate colored characters based on black and white silhouettes that can be loaded directly by browsing your files or using a random one generated with the option above, there are 2 possible outputs, the first is limited to just coloring the silhouette without adding any additional details while the second will use the information learnt by our network to create a colored version of the original draft but with some additions and quality improvements.
